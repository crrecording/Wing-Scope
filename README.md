# Wing Scope

**Wing Scope** is a utility for inspecting and monitoring Behringer WING console data in a clearer, faster way.

The goal is to provide a practical tool for live audio engineers who need quick visibility into WING status, routing, connectivity, and system behavior without digging through console menus during setup or show conditions.

## Purpose

Wing Scope is designed to help with:

* Viewing WING connection status
* Inspecting system and routing-related data
* Troubleshooting network or console communication issues
* Monitoring useful live-event information
* Giving engineers a clearer view of what the console is doing

Requirements may change, but the project may use:

* Node.js
* npm
* A Behringer WING console on the same network

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/wing-scope.git
cd wing-scope
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
